# 🦜 تطبيق توت (Toot App) - الإصدار v4.2

تطبيق "توت" هو رفيقك الذكي الذي يضع قوة الذكاء الاصطناعي في جيبك. صُمم ليكون المساعد الشخصي الأسرع والأكثر أماناً في العالم العربي.

## ✨ مميزات التطبيق
- **🚀 محرك صاروخي:** بفضل الانتقال لتقنية Groq (Llama 3.3)، ستحصل على الإجابات في أقل من ثانية.
- **🛡️ خصوصية مطلقة:** ميزة "وضع الضيف" تضمن عدم حفظ أي بيانات أو ملفات تعريف بعد خروجك من التطبيق.
- **🧠 ذكاء خارق:** قدرة عالية على البرمجة، حل المسائل المعقدة، والإبداع في الكتابة باللغتين العربية والإنجليزية.
- **🎨 تصميم عصري:** واجهة "Material 3" انسيابية تدعم الوضع الليلي والنهاري بشكل ذكي.

## 🆕 ما الجديد في v4.2؟
- وداعاً لمشاكل التعليق والبطء في Gemini؛ الآن نستخدم Groq API المستقر.
- إصلاح شامل لمشاكل تسجيل الدخول.
- تحسين أداء الواجهة لتوفير استهلاك البطارية.
