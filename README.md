## 🔥 لماذا "توت"؟

*   **⚡ سرعة البرق:** استجابة فورية بفضل تكامل محرك Groq المتطور.
*   **🤖 ذكاء لاما 3.3:** يستخدم التطبيق موديل `llama-3.3-70b` الذي يتفوق في فهم اللغة العربية والبرمجة.
*   **🔒 خصوصية مطلقة:** لا يتطلب التطبيق بيانات شخصية، مع دعم "وضع الضيف" لخصوصية كاملة.
*   **🎨 واجهة مستخدم مذهلة:** تصميم Material 3 عصري يوفر سلاسة تامة في التنقل.
