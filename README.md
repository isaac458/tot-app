# 🦜 تطبيق توت (Toot App) - الإصدار v4.3

تطبيق "توت" هو مساعدك الذكي الذي يضع قوة الذكاء الاصطناعي في جيبك. صُمم ليكون المساعد الشخصي الأسرع والأكثر أماناً في العالم العربي.

## ✨ مميزات التطبيق
- **🚀 محرك صاروخي:** بفضل الانتقال لتقنية Groq (Llama 3.3)، ستحصل على الإجابات في أقل من ثانية.
- **🛡️ خصوصية مطلقة:** ميزة "وضع الضيف" تضمن عدم حفظ أي بيانات أو ملفات تعريف بعد خروجك من التطبيق.
- **🧠 ذكاء خارق:** قدرة عالية على البرمجة، حل المسائل المعقدة، والإبداع في الكتابة باللغتين العربية والإنجليزية.
- **🎨 تصميم عصري:** واجهة "Material 3" انسيابية تدعم الوضع الليلي والنهاري بشكل ذكي.

## 🆕 ما الجديد في v4.3؟
- **تحسين استقرار الـ API:** إصلاح شامل لمشاكل الاتصال والـ IP.
- **بناء محلي آمن:** النسخة تتضمن مفتاح الـ API الخاص لضمان العمل الفوري.
- **واجهة مستخدم محسنة:** سلاسة أكبر في عرض رسائل الماركدوان (Markdown).
